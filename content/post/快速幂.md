快速幂正如其名字所说十分的快，非常的快，假如我们要求x^y，那么我们需要进行y次乘法，而快速幂却只需要进行y的二进制的1的个数次乘法就可以了。而且不光是快，快速幂还是能够在运算过程中取模。

```C++
int fastpow(int a,int b,int k){ //a^b%k
	int ans=1;
	while(b){
		if(b&1){
			ans*=a;
			ans%=k;
		}
		b=b>>1;
		a*=a;
		a%=k;
	}
	return a;
}
```
